Condicional switch

El condicional switch es una estructura de control que permite ejecutar diferentes bloques de código basándose en el valor de una expresión. Es especialmente útil cuando se tienen múltiples condiciones que evaluar.

Sintaxis

switch (expresión) {
    case valor1:
        // Código a ejecutar si expresión === valor1
        break;
    case valor2:
        // Código a ejecutar si expresión === valor2
        break;
    ...
    default:
        // Código a ejecutar si ninguna de las condiciones anteriores se cumple
}

Ejemplo

let fruta = 'manzana';
switch (fruta) {
    case 'manzana':
        console.log('Has elegido una manzana.');
        break;
    case 'banana':
        console.log('Has elegido una banana.');
        break;
    case 'naranja':
        console.log('Has elegido una naranja.');
        break;
    default:
        console.log('Fruta no reconocida.');
}

En este ejemplo, dependiendo del valor de la variable fruta, se ejecutará el bloque de código correspondiente. Si fruta es 'manzana', se imprimirá "Has elegido una manzana." en la consola.
