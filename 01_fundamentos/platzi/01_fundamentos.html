Fundamentos Javascript

Javascript

JavaScript es un lenguaje de programación que se utiliza principalmente para crear contenido interactivo en las páginas web. Fue desarrollado por Brendan Eich en 1995 y desde entonces se ha convertido en uno de los lenguajes de programación más populares del mundo.

JavaScript es un lenguaje de programación del lado del cliente, lo que significa que se ejecuta en el navegador web del usuario. Esto permite a los desarrolladores crear aplicaciones web dinámicas y ricas en funcionalidades, como formularios interactivos, animaciones, juegos y aplicaciones web completas.

JavaScript es un lenguaje de programación interpretado, lo que significa que no necesita ser compilado antes de ser ejecutado. En su lugar, el código JavaScript se interpreta en tiempo real por el navegador web del usuario.

JavaScript es un lenguaje de programación orientado a objetos, lo que significa que se basa en la creación de objetos y la manipulación de sus propiedades y métodos. Esto permite a los desarrolladores crear aplicaciones web complejas y escalables.

JavaScript es un lenguaje de programación muy versátil, que se puede utilizar para crear una amplia variedad de aplicaciones web, desde sitios web simples hasta aplicaciones web complejas y ricas en funcionalidades. Además, JavaScript se puede utilizar junto con otros lenguajes de programación, como HTML y CSS, para crear aplicaciones web completas y funcionales.

Variables

En JavaScript, una variable es un contenedor que se utiliza para almacenar datos o valores. Las variables se pueden utilizar para almacenar diferentes tipos de datos, como números, cadenas de texto, booleanos, objetos y arreglos.

Para declarar una variable en JavaScript, se utiliza la palabra clave var, let o const, seguida del nombre de la variable. Por ejemplo:

let nombre = "Juan";
const edad = 30;
var esEstudiante = true;

En este ejemplo, se declaran tres variables: nombre, edad y esEstudiante, que almacenan una cadena de texto, un número y un valor booleano, respectivamente.

Es importante tener en cuenta que las variables declaradas con let y const tienen un alcance de bloque, lo que significa que solo son accesibles dentro del bloque de código en el que se declaran. Por otro lado, las variables declaradas con var tienen un alcance de función, lo que significa que son accesibles en toda la función en la que se declaran.

Datos para variables

Primitivos

  • Número: Representa valores numéricos, tanto enteros como decimales. Ejemplo: let edad = 25;
  • Cadena de texto (String): Representa secuencias de caracteres. Ejemplo: let nombre = "Ana";
  • Booleano: Representa valores de verdad, es decir, true o false. Ejemplo: let esEstudiante = true;
  • Indefinido (Undefined): Representa una variable que ha sido declarada pero no asignada a ningún valor. Ejemplo: let x;
  • Nulo (Null): Representa la ausencia intencional de cualquier valor. Ejemplo: let y = null;
  • Símbolo (Symbol): Representa un valor único e inmutable, utilizado principalmente como identificadores de propiedades de objetos. Ejemplo: let sym = Symbol("id");
  • BigInt: Representa números enteros muy grandes que exceden el límite de los números normales en JavaScript. Ejemplo: let bigNumber = 123456789012345678901234567890n;

Compuestos

  • Objeto (Object): Representa una colección de propiedades y métodos. Ejemplo: let persona = { nombre: "Luis", edad: 28, profesión: "Ingeniero" };
  • Arreglo (Array): Representa una lista ordenada de valores. Ejemplo: let frutas = ["manzana", "banana", "naranja", "uva", "pera"];
  • Función (Function): Representa un bloque de código reutilizable. Ejemplo: function saludar() { console.log("Hola"); }

Manipulación de Strings

Las cadenas de texto (strings) en JavaScript son secuencias de caracteres que se utilizan para representar texto. Se pueden manipular de diversas formas utilizando métodos incorporados en el lenguaje. A continuación, se presentan algunos ejemplos comunes de manipulación de strings:

  • Concatenación: Combinar dos o más cadenas de texto utilizando el operador + o el método concat(). Ejemplo: 
    let saludo = "Hola, ";
                let nombre = "María";
                let mensaje = saludo + nombre; // "Hola, María"
                // o
                let mensaje2 = saludo.concat(nombre); // "Hola, María"
                // o
                let mensaje3 = `${saludo}${nombre}`; // "Hola, María"
  • Longitud: Obtener la longitud de una cadena utilizando la propiedad length. Ejemplo: 
    let texto = "JavaScript";
                let longitud = texto.length; // 10
  • Subcadena: Extraer una parte de una cadena utilizando los métodos substring(), substr() o slice(). Ejemplo: 
    let frase = "Aprendiendo JavaScript";
                let subcadena1 = frase.substring(0, 11); // "Aprendiendo"
                let subcadena2 = frase.substr(12, 10); // "JavaScript"
                let subcadena3 = frase.slice(0, 11); // "Aprendiendo"
  • Reemplazo: Reemplazar parte de una cadena utilizando el método replace(). Ejemplo: 
    let mensaje = "Hola Mundo";
                let nuevoMensaje = mensaje.replace("Mundo", "JavaScript"); // "Hola JavaScript"
  • Conversión a mayúsculas/minúsculas: Convertir una cadena a mayúsculas o minúsculas utilizando los métodos toUpperCase() y toLowerCase(). Ejemplo: 
    let texto = "JavaScript";
                let mayusculas = texto.toUpperCase(); // "JAVASCRIPT"
                let minusculas = texto.toLowerCase(); // "javascript"
  • División: Dividir una cadena en un arreglo de subcadenas utilizando el método split(). Ejemplo: 
    let frase = "Aprendiendo JavaScript es divertido";
                let palabras = frase.split(" "); // ["Aprendiendo", "JavaScript", "es", "divertido"]

Números y operaciones

En JavaScript, los números se representan utilizando el tipo de dato Number, que puede manejar tanto números enteros como decimales.

Tipos de números

  • Números enteros (Integer): Son números sin parte decimal, como 1, -5, 1000.
  • Números de punto flotante (Floating-point): Son números con parte decimal, como 3.14, -0.5, 2.718.
  • Números especiales: JavaScript también tiene valores especiales para representar conceptos matemáticos como Infinity, -Infinity y NaN (Not a Number).
  • Números con notación científica (Exponential notation): Son números representados en la forma de un número base multiplicado por una potencia de 10. Ejemplo: 1.23e4 representa 1.23 × 10^4 (12300).

Operaciones

Operaciones aritméticas

  • Suma (+): Suma dos números. Ejemplo: 5 + 3 // 8
  • Resta (-): Resta un número de otro. Ejemplo: 10 - 4 // 6
  • Multiplicación (*): Multiplica dos números. Ejemplo: 6 * 7 // 42
  • División (/): Divide un número por otro. Ejemplo: 20 / 5 // 4
  • Módulo (%): Devuelve el resto de la división entre dos números. Ejemplo: 10 % 3 // 1
  • Exponenciación (**): Eleva un número a la potencia de otro. Ejemplo: 2 ** 3 // 8

Operaciones de precisión

  • Redondeo: Utiliza métodos como Math.round(), Math.floor() y Math.ceil() para redondear números. Ejemplo: 
    Math.round(4.5); // 5
Math.floor(4.9); // 4
Math.ceil(4.1); // 5
  • Formato decimal: Utiliza el método toFixed() para formatear un número con un número específico de decimales. Ejemplo: 
    let num = 3.14159;
let formattedNum = num.toFixed(2); // "3.14"

Operaciones avanzadas

  • Generación de números aleatorios: Utiliza Math.random() para generar un número aleatorio entre 0 y 1. Ejemplo: 
    let randomNum = Math.random(); // Número aleatorio entre 0 y 1
  • Funciones matemáticas: JavaScript proporciona varias funciones matemáticas a través del objeto Math, como Math.sqrt() (raíz cuadrada), Math.pow() (potencia), Math.max() (máximo) y Math.min() (mínimo). Ejemplo: 
    let raiz = Math.sqrt(16); // 4
let potencia = Math.pow(2, 3); // 8
let maximo = Math.max(5, 10, 3); // 10
let minimo = Math.min(5, 10, 3); // 3

Type casting

El type casting en JavaScript se refiere al proceso de convertir un valor de un tipo de dato a otro. JavaScript es un lenguaje de tipado dinámico, lo que significa que las variables pueden cambiar de tipo en tiempo de ejecución. Existen dos tipos principales de type casting: implícito y explícito.

Tipos de lenguaje para convertir

Compilados

Traducción y conversión de tipos en lenguajes compilados suelen ser explícitas y controladas por el programador. Son traducidos a código máquina antes de la ejecución.

Chequeo estático de tipos: Los lenguajes compilados suelen realizar un chequeo de tipos en tiempo de compilación para detectar errores de tipo antes de ejecutar el programa.

  • C, C++, Rust, Go, Java, C#, Swift: Utilizan conversiones explícitas mediante funciones o métodos específicos para convertir entre tipos de datos.
  • Ejemplo en C++: 
    int numero = 10;
double decimal = static_cast<double>(numero); // Conversión explícita de int a double

Interpretados

Traducción y conversión de tipos en lenguajes interpretados suelen ser más flexibles y dinámicas. La conversión de tipos puede ocurrir en tiempo de ejecución y a menudo es implícita.

Chequeo dinámico de tipos: Los lenguajes interpretados suelen realizar un chequeo de tipos en tiempo de ejecución, lo que permite mayor flexibilidad pero puede introducir errores en tiempo de ejecución si no se manejan adecuadamente.

  • JavaScript, Python, Ruby, PHP: Permiten tanto conversiones implícitas como explícitas entre tipos de datos.
  • Ejemplo en JavaScript: 
    let numero = 10;
let decimal = Number(numero); // Conversión explícita de int a float

Type casting implícito

El type casting implícito ocurre cuando JavaScript convierte automáticamente un valor de un tipo a otro sin que el programador lo indique explícitamente. Esto suele suceder en operaciones que involucran diferentes tipos de datos. Algunos ejemplos comunes de type casting implícito son:

  • Cadenas: Cuando hay una concatenación entre una cadena y otro tipo de dato, JavaScript convierte automáticamente el otro tipo a cadena. Ejemplo: 
    let resultado = "El número es: " + 5; // "El número es: 5"
    let texto = "Hola" + 123; // "Hola123"
    let saludo = "Hola" + true; // "Holatrue"
  • Números: Cuando no hay cadenas involucradas, JavaScript convierte automáticamente los valores a números. Ejemplo: 
    let suma = true + 1; // 2 (true se convierte a 1)

Type casting explícito

El type casting explícito ocurre cuando el programador indica de manera clara y directa que desea convertir un valor de un tipo a otro.

  • Conversión a número: Utilizando la función Number(), el operador unario + o parseInt() y parseFloat(). Ejemplo: 
    let numero1 = Number("42"); // 42
let numero2 = +"3.14"; // 3.14
let entero = parseInt("100"); // 100
let decimal = parseFloat("3.14"); // 3.14
  • Conversión a cadena: Utilizando la función String() o el método toString(). Ejemplo: 
    let texto1 = String(123); // "123"
let texto2 = (456).toString(); // "456"
  • Conversión a booleano: Utilizando la función Boolean() o mediante doble negación !!. Ejemplo: 
    let bool1 = Boolean(1); // true
let bool2 = !!0; // false

Operaciones de comparación

En JavaScript, las operaciones de comparación se utilizan para comparar dos valores y determinar su relación. Estas operaciones devuelven un valor booleano (true o false) según el resultado de la comparación. A continuación, se describen las principales operaciones de comparación en JavaScript:

  • Igualdad estricta (===): Compara dos valores y sus tipos. Devuelve true si ambos son iguales en valor y tipo, de lo contrario devuelve false. Ejemplo: 
    5 === 5; // true
5 === '5'; // false
  • Igualdad débil (==): Compara dos valores después de realizar conversiones de tipo si es necesario. Devuelve true si los valores son iguales después de la conversión, de lo contrario devuelve false. Ejemplo: 
    5 == '5'; // true
0 == false; // true
  • Desigualdad estricta (!==): Compara dos valores y sus tipos. Devuelve true si los valores son diferentes en valor o tipo, de lo contrario devuelve false. Ejemplo: 
    5 !== 10; // true
5 !== '5'; // true
  • Desigualdad débil (!=): Compara dos valores después de realizar conversiones de tipo si es necesario. Devuelve true si los valores son diferentes después de la conversión, de lo contrario devuelve false. Ejemplo: 
    5 != '10'; // true
5 != '5'; // false
  • Mayor que (>): Devuelve true si el valor de la izquierda es mayor que el valor de la derecha, de lo contrario devuelve false. Ejemplo: 
    10 > 5; // true
3 > 7; // false
  • Menor que (<): Devuelve true si el valor de la izquierda es menor que el valor de la derecha, de lo contrario devuelve false. Ejemplo: 
    3 < 7; // true
10 < 5; // false
  • Mayor o igual que (>=): Devuelve true si el valor de la izquierda es mayor o igual que el valor de la derecha, de lo contrario devuelve false. Ejemplo: 
    7 >= 7; // true
10 >= 5; // true
  • Menor o igual que (<=): Devuelve true si el valor de la izquierda es menor o igual que el valor de la derecha, de lo contrario devuelve false. Ejemplo: 
    5 <= 10; // true
7 <= 3; // false

Operadores lógicos

En JavaScript, los operadores lógicos se utilizan para combinar o modificar valores booleanos (true o false). Estos operadores permiten realizar operaciones lógicas y tomar decisiones basadas en múltiples condiciones. A continuación, se describen los principales operadores lógicos en JavaScript:

  • AND lógico (&&): Devuelve true si ambos operandos son true, de lo contrario devuelve false. Ejemplo: 
    true && true; // true
true && false; // false
false && false; // false
  • OR lógico (||): Devuelve true si al menos uno de los operandos es true, de lo contrario devuelve false. Ejemplo: 
    true || false; // true
false || false; // false
  • NOT lógico (!): Invierte el valor booleano del operando. Devuelve true si el operando es false, y devuelve false si el operando es true. Ejemplo: 
    !true; // false
!false; // true
